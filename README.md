# Mantine + Vite + Vanilla extract template

A design system project using Mantine UI components with Vanilla Extract for styling and Ladle for component documentation.

## Getting Started

### Install Dependencies

```bash
yarn install
```

## Running the Project


### Ladle (Component Storybook) 

Ladle is used to showcase and document your design system components.

#### Development Mode

Run Ladle in development mode with hot reload:

```bash
yarn ladle
```

This starts Ladle at `http://localhost:61000/` (or another available port)

#### Build for Production

Build the Ladle static site:

```bash
yarn ladle:build
```

#### Preview Production Build

Preview the built Ladle site:

```bash
yarn ladle:preview
```
### Main Application

Run the main Vite development server: (currently empty)

```bash
yarn dev
```

This starts the main application at `http://localhost:5173/`
